# TP2-TrabajoFinal
Trabajo Final de Taller de Programacion 2

## Variables de entorno
Crea un archivo `.env` en la raíz del proyecto con las siguientes variables:

```env
PORT=3000
DB_HOST=localhost
DB_PORT=3306
DB_NAME=biblioteca_db
DB_USER=root
DB_PASSWORD=contoso
```

El proyecto ya carga estas variables usando `dotenv` en `src/config/config.js`.
